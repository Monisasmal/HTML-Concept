HTML Media & Responsive Images Q&A

  • Q41 — <embed>, <object>, <iframe> differences?

    <iframe> → embeds another HTML page (like YouTube, Google Maps).
    <object> → can embed multimedia & supports fallback content.
    <embed> → embeds external content (simpler, plugin-based like PDF).

  • Q42 — Responsive images: srcset and sizes?

    srcset → provides images in different resolutions/sizes
    sizes → tells browser how much space the image will take
    Browser chooses best file to save bandwidth & improve performance.

  • Q43 — <picture> vs <img>?

    <picture> → art direction — different images for different screen sizes/devices.
    <img> → single source with optional srcset for resolution selection.
